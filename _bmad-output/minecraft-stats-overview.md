# Minecraft Stats — Ce qui s'en vient

> Le site de stats que vous connaissez déjà, transformé en **la** référence pour trouver un serveur Minecraft.

---

## Le problème avec les sites de vote actuels

Les sites comme minecraft-vote.fr ou serveur-minecraft-vote.fr existent depuis des années, mais ils ont tous les mêmes défauts :
- Design daté et peu agréable
- Aucune donnée réelle sur les serveurs — juste des votes
- Certains serveurs affichent de faux nombres de joueurs connectés
- Aucun intérêt à revenir après avoir voté une fois

**Minecraft Stats change ça.**

---

## Ce qui arrive

### Vote communautaire
Les joueurs pourront voter pour leurs serveurs préférés, directement depuis le site — **sans créer de compte**. Les serveurs sont classés par :
- Votes du mois (remis à zéro le 1er — tout le monde repart à zéro !)
- Votes de l'année
- Votes All Time

### Vraies statistiques + votes = confiance
Contrairement aux concurrents, Minecraft Stats combine les votes avec les **vraies courbes de joueurs** sur plusieurs mois. Tu peux voir si un serveur est réellement actif, pas juste bien organisé pour voter.

### Badge "Vérifié"
Les serveurs qui installent notre plugin affichent un badge **Vérifié** — ça confirme que leur nombre de joueurs connectés est réel, pas inventé. C'est la première fois que ça existe sur un site de vote Minecraft.

### Tendances du moment
Une section dédiée aux serveurs qui montent fort cette semaine — parfait pour découvrir les nouveaux serveurs en pleine croissance avant tout le monde.

### Récompenses pour les votants *(à venir)*
Les admins de serveurs pourront configurer des récompenses in-game pour les joueurs qui votent : items, grades, tokens... Plus tu votes régulièrement, plus ton cooldown entre les votes diminue.

### Deux façons d'intégrer le système
Pour les admins de serveurs, deux options simples :
- **Plugin officiel** (open source) — installe et c'est bon
- **API HTTP** — pour ceux qui préfèrent gérer ça eux-mêmes dans leur propre code

---

## Pour les admins de serveurs

- Revendiquer son serveur existant en 2 minutes (vérification DNS)
- Configurer le cooldown de vote (1h, 6h, 12h, 24h...)
- Définir des paliers de récompenses pour motiver les joueurs
- Voir les statistiques de votes en temps réel
